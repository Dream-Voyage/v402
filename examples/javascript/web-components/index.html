V402 Payment Button

Web Component Integration Example

Basic Usage

The V402 Payment Button is a custom web component that can be easily integrated into any web application. It provides a complete payment interface for v402 protocol products. 

<v402-payment-button
    product-id="product-123"
    api-url="https://api.v402.network"
    public-key="0x1234567890abcdef1234567890abcdef12345678"
    private-key="0xabcdef1234567890abcdef1234567890abcdef1234567890abcdef1234567890"
></v402-payment-button>

Note:

In a real application, you would use actual product IDs and cryptographic keys. This example uses mock data for demonstration purposes.

Event Handling

The component dispatches custom events that you can listen to: 

const paymentButton = document.querySelector('v402-payment-button');

paymentButton.addEventListener('payment-success', (event) => {
    console.log('Payment successful:', event.detail);
    // Handle successful payment
});

paymentButton.addEventListener('payment-error', (event) => {
    console.log('Payment failed:', event.detail);
    // Handle payment error
});

Integration with Frameworks

The web component can be used with any framework or vanilla JavaScript: 

// React
function PaymentPage() {
    const handlePaymentSuccess = (event) => {
        console.log('Payment successful:', event.detail);
    };
    
    return (
        <v402-payment-button
            product-id="product-123"
            onPaymentSuccess={handlePaymentSuccess}
        />
    );
}

// Vue
<template>
    <v402-payment-button
        product-id="product-123"
        @payment-success="handlePaymentSuccess"
    />
</template>

// Angular
<v402-payment-button
    product-id="product-123"
    (payment-success)="handlePaymentSuccess($event)"
></v402-payment-button>
